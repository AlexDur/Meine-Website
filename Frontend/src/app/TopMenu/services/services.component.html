Fullstack-Entwicklung

Im Laufe der Jahre bin ich durch mein berufliches Umfeld zum Full-Stack-Entwickler geworden. Dadurch gelingt es mir ganze Anwendungen eigenständig zu programmieren und zu veröffentlichen.

  • Frontend: Am Anfang steht der Entwurf eines ansprechenden Designs für die Anwendung. Im Frontend wird dieser Entwurf in Code umgesetzt. Das Ergebnis ist eine für den Nutzer sichtbare Oberfläche mittels der er mit der Anwendung kommunizieren kann.
  • Backend: Das Backend ist der für den Nutzer unsichtbare Teil einer Anwendung. Er beinhaltet die Logik, Datenbanken und die Verarbeitung von Daten und kommuniziert mit dem Frontend. Erst durch das Backend wird aus einer ansprechenden Oberfläche eine lebendige, vollfunktionale Anwendung.
