Mein Techstack

Für die Entwicklung in beiden Bereichen werden jeweils verschiedene Technologien eingesetzt, deren Gesamtheit den sogenannten Techstack darstellt. Mit dem aufgeführten Techstack ist eine Umsetzung der allermeisten Webanwendungen möglich.

Unter "Tools" sind außerdem noch Technologien und Methoden angeführt, die nützliche Helfer für die organisatorischen und konzeptuellen Aufgaben sind.

Mein Techstack zur Umsetzung von Projekten umfasst die folgenden Technologien:

HTML/SCSS  |  JavaScript  |  Angular + TypeScript   |   Playwright  |  Barrierefreiheit

Java  |  Spring Boot  |  Node.js  |  Rest APIs  |  OpenAPI  |  MySQL  |  Liquibase  |  AWS  |  Docker

JIRA  |  Confluence  |  UML  |  FIGMA  |  GIMP
