Mein Techstack

Zur Entwicklung in beiden Bereichen werden jeweils verschiedene Technologien eingesetzt, deren Gesamtheit den sogenannten Techstack darstellt.

Mein Techstack zur Umsetzung von Projekten umfasst die folgenden Technologien:

HTML/SCSS  |  JavaScript  |  Angular + TypeScript   |   Playwright  |  Figma  |  Barrierefreiheit

Java  |  Spring Boot  |  Node.js  |  Rest APIs  |  OpenAPI  |  MySQL  |  Liquibase  |  AWS  |  Docker
